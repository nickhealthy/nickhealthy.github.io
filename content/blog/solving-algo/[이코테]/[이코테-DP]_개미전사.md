## My Solution

```python
n = int(input())
arr = list(map(int, input().split()))
dp_table = [0] * 100

dp_table[0] = arr[0]
dp_table[1] = max(arr[0], arr[1])

for i in range(2, n):
    dp_table[i] = max(arr[i], arr[i - 2] + arr[i])

print(dp_table[n - 1])
```
