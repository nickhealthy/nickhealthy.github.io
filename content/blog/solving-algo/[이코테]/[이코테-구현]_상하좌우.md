## My Solution

```python
n = int(input())
move_list = input().split()

x, y = 1, 1
dx = [0, 0, -1, 1]
dy = [-1, 1, 0, 0]
move_type = ['L', 'R', 'U', 'D']

for move_lst in move_list:
    for i in range(len(move_type)):
        if move_lst == move_type[i]:
            nx = dx[i] + x
            ny = dy[i] + y

    if nx < 1 or ny < 1 or nx > n or ny > n:
        continue

    x, y = nx, ny

print(x, y)
```
