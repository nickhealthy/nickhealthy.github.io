## My Solution

```python
def solution(n) :
    answer = 0
    lst = [0 for i in range(n)]
    lst[0] = 1
    lst[1] = 2
    for i in range(2, n) :
        lst[i] = (lst[i - 1] + lst[i - 2]) % 1000000007
    answer = lst[n - 1]
    return answer
```
