## My Solution 2

```python
def arrayPairSum(nums):
    sum = 0
    nums.sort()

    for i, n in enumerate(nums):
        if i % 2 == 0:
            sum += n

    return sum
```

def arrayPairSum(nums):
    return sum(sorted(nums)[::2])
